## Chronostasis

### 作者

victifa

### 方向

Misc / Blockchain

### 知识点

- UniswapV2 TWAP 预言机机制
- 异步 LP 金库（EIP-7540）
- 闪电贷与价格操纵
- 时间加权平均价格快照与赎回清算的时间差利用

### 难度

中级

### 内容

使用异步 LP 金库和 TWAP 定价。金库在赎回请求时快照份额价格，在清算时按当前 LP 价格结算。选手需要让金库内 LP 余额低于初始水平。

选手持有 TKA、TKB、TKC 三种代币，可通过 UniswapV2 池交易、存入金库获取份额、提交异步赎回请求。协议包含两个交易池，以及一个 TWAP 预言机为 LP 公允价值报价。
