# BrainfOok

## 基本介绍

本题目实现了一种魔改自Brainfuck的VM，基于python编写，除了实现难度简单、反编译难度低之外，还充分地利用了python**执行速度慢**的“优秀”特点和几个数组特性，比如**负索引**和**大整数**等，这是本题的第一个坑点：由于python支持负索引，因此盲目地将代码逻辑转为C/C++会产生许多逻辑错误，必须考虑到python的数组特性才能正确模拟。虚拟机自身有4个寄存器、96个内存槽、2个32位比特内存和两套互不干扰的无限大的跳转地址保存栈。除比特内存外，其他均等价于由正常python数组实现，因此数值无上下限（由于复杂度限制，本题并未用到大整数）。VM共有25种指令，分类如下：

| opcode | 作用 |
| --- | --- |
| +- | inc/dec当前寄存器 |
| <> | 切换寄存器指针 |
| ] | 保存寄存器值作为内存指针 |
| {} | 从内存复制值/保存寄存器值到内存 |
| ? | 与0比较，为0设置zero flag为true，否则为false |
| !; | 设置代码回溯起点/若zero flag为假则回溯到上一个!，否则继续执行 |
| #() | 寻找下游的()，将当前寄存器值作为跳转偏移量，计算基于()的跳转地址/jz/jnz |
| :. | 输入/输出 |
| $@[ | 将寄存器值存入比特内存/从比特内存弹出到寄存器/切换比特寄存器 |
| ^&|~ | xor/and/or/not |
| ,* | 将当前比特内存从低位连续,个的位置0/将当前寄存器值开始的索引直到最高位置0 |
| % | 结束 |

由于VM指令中能进行控制流分支的opcode只有`?`一种，且只能和0比较，因此当输入不当时这个VM非常容易陷入死循环，比如对`-1`持续减一：`!-?;`，死循环也用于替代Wrong出现在后半段的校验逻辑中。此外，由于BrainfOok的基本运算符显著少于汇编类型，基本的加减乘除都没有，也就是**BrainfOok的运算层级低于汇编**，可以通过**自由搭配**自身的opcode实现基本的汇编运算，因此VM实现opcode中没有的运算的方式并不固定，那么想要解这道题目，不能靠传统的**“转汇编并trace”**思路来理解，**不存在统一的模板能转成对应的汇编运算**，且由于实现各种运算的复杂度过高，**单步opcode操作信息密度过低**，trace也不是一个好的选择。真正要解出这道题，要从BrainfOok本身的角度来理解题目，首先要学会写BrainfOok的自身代码，只有理解了BrainfOok的运行逻辑，才能继续考虑解题。以下这些是一些运算操作的模板（实际题目代码中还存在大量变种/其他实现方式）：

```
!-?; / !+?;                                eset register to 0
!<+>-?; / !<->-?;                          add / sub reg0, reg1
!<+>-?; / !>+<-?;                          move to previous / next register
>++++!<+++>-?;                             mul num2 with num1 and save the result to reg0
!>{!>+<-?;<-?;                             mul reg_n with mem and save result to reg_n+2
+++->{!!>{!>+<-?;<-?;>>!<<+>>-?;<<<-?>;    pow arg now with instant num (num to reg0, arg to reg1)
!---?>+<;                                  div reg0 with instant num, no residue
$,,,@!--------?>+<;                        shr reg
```

因为缺少对>0和<0情况的区分opcode，实现除法将是一场灾难，实际上，上文中的除法仅能应对整除的情况，对于shr有特殊的`,`运算符来协助实现，而对于一般的带余除法则非常难以实现，大致逻辑为：

```python
def div(dividend, divisor):
    quotient = 0
    while True:
        dividend -= divisor
        quotient += 1
        if dividend == 0: residue = 0; return quotient, residue
        else:
            residue = divisor
            for i in range(divisor-1):
                dividend += 1
                residue -= 1
                if dividend == 0: return quotient-1, residue
            dividend -= divisor-1
```

此外，由于VM会自动过滤不在指令集中的字符，因此空格和字母等是合法的junk code，因此可以在code中塞入大量无关字符（A-Za-z0-9_和空格）。

## 解混淆

在去掉外层的字符画、特殊编码和AES加密后，会得到掺杂了大量垃圾字符的混淆code，去掉垃圾字符后即可，留下真正会被执行的纯代码（这里也可以打开程序后直接从内存拿）。

首先，注意到有一些opcode之间的作用会相互抵消：比如+-，<>，<<<<，>>>>，[[等，因此插入大量对称的opcode后逻辑可以保持不变，这是第一种混淆类型，我们称之为“对称运算”；

其次，还有一些安全的冗余位操作，比如在不需要进行位运算的时候不断地压寄存器值入比特内存、切换比特内存指针、进行位运算等，只要保证补充比特内存加载值到寄存器即可，这是第二种混淆类型，我们称之为“无效位运算”；

此外，还有一些无效的内存操作，在每次的]/{}之后的区域立即插入]/{}不会改变任何内容，因此可以有大量连续]、{}等，我们称之为“无效内存操作”。

以上三种就是本题用到的所有混淆类型，可以直接识别这三种类型的相关模式来进行解混淆，**但同时必须要注意**，`#()`相关的运算符会计算并偏移量进行跳转，因此强制解混淆几乎必定会导致跳转地址出错。想要安全地解混淆，需要对混淆代码的每个会执行跳转的`()`进行watch，在执行到其位置且判断将要跳转时，记录其跳转的偏移量，将从`()`开始的offset范围内设置为保留区域，仅能对非保留区域的代码进行解混淆。但BrainfOok的**执行速度极慢**，随着解混淆进度的推进，`()`的位置只会越来越靠后，后期的steps可能会达到数亿，这会非常影响解混淆的速度和题目逻辑的分析，因此除了在确认BrainfOok要用到的python特性后将其转为C/C++代码这种方式，我们还可以进行**带断点续跑的分段trace**：记录运行到某个位置时的寄存器与内存布局、跳转栈、标志位等全部信息，然后直接修改运行起始代码地址，从记录点恢复续跑，这样就能防止重复跑不必要的代码区域，从而快速得到每个`()`跳转的具体偏移量，从而能一步步完成整个解混淆过程。

实际上，题目的校验逻辑和解混淆逻辑是同步的，都需要分段运行并分段考虑，因为后续的校验可能会用到前面算出来的中间过程量，能对`()`进行偏移量的收集也依赖代码本身能运行到其所在的位置（即完成前置所有的校验），因此**题目不能一步恢复所有代码**，必须逐级递进，**不存在**从一开始就能单次解所有混淆和解所有约束的脚本。

以下是可从题目中逆向得到的虚拟机运行函数：

```python
def print_stat(stat):
    buf = bytearray()
    for byte_val in stat:
        if byte_val == 0: break
        elif 32 <= byte_val <= 126: buf.append(byte_val)
    result_str = buf.decode('ascii', errors='ignore')
    print(result_str)

def itb(n, bsch, bst1, bst2):
    bst1 = list(bst1)
    bst2 = list(bst2)
    target = bst1 if not bsch else bst2
    n_bits = n.bit_length()
    for i in range(len(target)):
        if i < n_bits: target[i] = (n >> i) & 1
        else: target[i] = 0
    if not bsch: bst1 = target
    else: bst2 = target
    return bst1, bst2

def bti(bsch, bst1, bst2):
    ba = bst1 if not bsch else bst2
    res = 0
    for i, bit in enumerate(ba):
        if bit: res |= (1 << i)
    return res

def bit_and(bst1, bst2): return [a & b for a, b in zip(bst1, bst2)]

def bit_or(bst1, bst2): return [a | b for a, b in zip(bst1, bst2)]

def bit_xor(bst1, bst2): return [a ^ b for a, b in zip(bst1, bst2)]

def bit_not(bsch, bst1, bst2):
    target = bst1 if not bsch else bst2
    target = [0 if bit else 1 for bit in target]
    if not bsch: bst1 = target
    else: bst2 = target
    return bst1, bst2

def bit_pth(bsch, bst1, bst2, t):
    target = bst1 if not bsch else bst2
    for i in range(t): target[i] = 0
    if not bsch: bst1 = target
    else: bst2 = target
    return bst1, bst2

def bit_bool(bsch, bst1, bst2, bptr):
    target = bst1 if not bsch else bst2
    return True if target[bptr] else False

def bit_low(bsch, bst1, bst2, idx):
    target = bst1 if not bsch else bst2
    for i in range(idx, len(target)): target[i] = 0
    if not bsch: bst1 = target
    else: bst2 = target
    return bst1, bst2

def vm(input_queue, output_queue):
    size = 96          # memory size
    stat = [0] * size  # memory stat
    sptr = 0           # stat pointer
    cptr = 0           # code pointer
    cpts = []          # saved code pointers
    regs = [0] * 4     # registers
    idxr = 0           # index of register
    zflg = False       # ZeroFlags
    bst1 = [0] * 32    # bit stack 1
    bst2 = [0] * 32    # bit stack 2
    bsch = False       # bit stack choice
    bptr = 0           # bit stack bit pointer
    step = 0           # steps
    # 选择输入模式
    # cint = 0           # continue input tag, for manual input
    # inbf = ""          # input buffer, for manual input
    cint = 2           # continue input tag, for auto input
    inbf = "XXX" # input buffer, for auto input
    code = "XXX".join(char for char in code if char in "+-<>!?;{}[]().:$@&|^~,*%#")
    st = time.time()
    while cptr < len(code):
        opcd = code[cptr]
        if opcd == ">": idxr = (idxr + 1) % 4              # add index of register
        if opcd == "<": idxr = (idxr - 1) % 4              # sub index of register
        if opcd == "]": sptr = regs[idxr]                  # save stat addr
        if opcd == "{": regs[idxr] = stat[sptr]            # load stat value
        if opcd == "}": stat[sptr] = regs[idxr]            # save stat value
        if opcd == "+": regs[idxr] += 1                    # self inc1
        if opcd == "-": regs[idxr] -= 1                    # self dec1
        if opcd == "!": cpts.append(cptr)                  # load code position
        if opcd == "#":
            t = 0
            while cptr + t < len(code) - 1:
                t += 1
                if code[cptr + t] in "()":
                    cpts.append(cptr + t + regs[idxr])     # save jmp addr
                    break
        if opcd == "?": zflg = regs[idxr] == 0             # cmp with 0
        if opcd == ";":                                    # jnz to stack saved value (for do-while)
            if zflg == False: cptr = cpts[-1]
            else:
                del cpts[-1]
                if cint == 1: cint = 0                     # end continue input (instring)
        if opcd == "(":                                    # jz to reg offset
            if zflg == True: print("cptr: ",cptr, "off: ", cpts[-1]-cptr); cptr = cpts[-1]
            del cpts[-1]
        if opcd == ")":                                    # jnz to reg offset
            if zflg == False: print("cptr: ",cptr, "off: ", cpts[-1]-cptr); cptr = cpts[-1]
            del cpts[-1]
        if opcd == ":":                                    # input
            if cint == False: inbf = input()               # inbuffer
            if code[cptr-1] == "!": cint = 1               # instring
            else: inbf = inbf[0]                           # inchar
            if not len(inbf) == 0: regs[idxr] = ord(inbf[0]); inbf = inbf[1:]
        if opcd == ".": print_stat(stat)                                   # print
        if opcd == "$": bst1, bst2 = itb(regs[idxr], bsch, bst1, bst2)     # load to bit stack
        if opcd == "@": regs[idxr] = bti(bsch, bst1, bst2)                 # pop from bit stack
        if opcd == "[": bsch = not bsch                                    # change bit stack ptr
        if opcd == "&": bst1 = bit_and(bst1, bst2)                         # and
        if opcd == "|": bst1 = bit_or(bst1, bst2)                          # or
        if opcd == "^": bst1 = bit_xor(bst1, bst2)                         # xor
        if opcd == "~": bst1, bst2 = bit_not(bsch, bst1, bst2)             # not
        if opcd == ",":
            t = 0
            while cptr + t < len(code) - 1:
                t += 1
                if code[cptr + t] != ",": break
            bst1, bst2 = bit_pth(bsch, bst1, bst2, t)                      # bits patch
        if opcd == "*": bst1, bst2 = bit_low(bsch, bst1, bst2, regs[idxr]) # low half bits
        if opcd == "%": print("EOF"); break                                # EOF
        cptr += 1
        step += 1
        if step % 1000000 == 0: print("step: ", step)

    et = time.time()

    # 调试打印
    print(code)
    print("len = ", len(code))
    print_stat(stat)
    print("stat =", stat)
    print("regs =", regs)
    print("bst1 =", bst1, "\nbst2 =", bst2)
    print("cptr =", cptr, "\nidxr =", idxr, "\nsptr =", sptr, "\nzflg =", zflg, "\nbsch =", bsch, "\nbptr =", bptr)
    print("cpts =", cpts)
    print("steps:", step, "time:", et-st)
```
